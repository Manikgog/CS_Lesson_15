Задание: Реализация игры "Крестики-нолики" с использованием интерфейсов.

Описание: Вам необходимо создать консольную игру "Крестики-нолики" для двух игроков. Игроки должны вводить координаты своего хода, и игра должна проверять их на корректность и определять победителя. Для проверки корректности хода используйте интерфейс.

Требования:

Создайте классы Board (доска) и Player (игрок).

Интерфейс IBoard должен содержать следующие методы:

void Initialize() - инициализирует игровую доску.
void Draw() - отрисовывает текущее состояние доски в консоли.
bool IsGameOver(out string winner) - проверяет, является ли игра завершенной.
bool IsValidMove(int row, int col) - проверяет, является ли ход (row, col) корректным.
void MakeMove(int row, int col, char symbol) - делает ход игрока с символом symbol (X или O).
Класс Player должен иметь следующие свойства и методы:

string Name - имя игрока.
char Symbol - символ, которым игрок будет обозначаться на доске.
void MakeMove(IBoard board) - запрашивает у пользователя координаты хода и передает их в метод MakeMove интерфейса IBoard.
Реализуйте класс Board, который реализует интерфейс IBoard. Добавьте необходимые поля, свойства и методы для хранения и обработки состояния доски.

Создайте главный класс Game, в котором будет происходить инициализация игры, а также цикл ввода ходов игроков и проверки на завершение игры.

Для определения победителя введите следующие условия:

Три одинаковых символа в одной строке.
Три одинаковых символа в одном столбце.
Три одинаковых символа на диагонали.
Обратите внимание на обработку ошибок при вводе некорректных координат и проверку на ничью.

Используйте принципы ООП, такие как инкапсуляция, наследование, полиморфизм, где это применимо.

Не забудьте комментировать свой код, чтобы сделать его понятным и читаемым.
